*** data/ ***

This is the data directory containing all data files for running 
the SNP analysis. Data files are **NOT** included in the git repo.
Data files can be generated by running the scripts in order (by number).

RNAseq and miRNAseq data can be generated by following the repo at:
https://github.com/gawilk/miRNA-pathways

SNP data can only be downloaded with permission from TCGA, or from another resource.
